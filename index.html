学习路上的坑

bug

不要轻易的放弃,因为当我回头的时候才发现,那些坑我还是趴过来,时间还是在前行

display:block,inline

块级标签：独占一行，宽度默认为父级标签的宽度，可以改变宽度
行级标签：不会独占一行，宽度由内容来决定，不能改变宽高

在我学习的时候或者在练习网页的时候很喜欢在行级标签的时候把它当做块级来用，所用每次都在做先无用功，效果都没有

margin

html如下 
    <div class='box1'>box1
        <div class='box2'>box2</div>
    </div>    
css 如下
.box1{height:100px;width:200px;background:red}
.box2{height:80px;width:100px;background:pink}

opacity：0~1     半透明效果,1:完全不透明；0:完全透明，(看不到)。

<div class="box">
    <div class="content"></div>
</div>

.box{
    width:400px;
    height:600px;
    background-color:black;
    position:relative;
    opacity:0.1;
}
.content{
    width:200px;
    height:350px;
    background-color:red;
    position:absolute;
    top:50%;
    left:50%;
    margin-left:-100px;
    margin-top:-175px;
}

需要注意：opacity 不仅让标签自身变成半透明，同时标签的子元素也会变成半透明的。
所以opacity要慎用！尽量用rgba来设置背景半透明。
