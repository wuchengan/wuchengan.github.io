gulp自动化构建工具

##gulp安装
1.创建项目文件
2.进入项目文件
3.运行 npm init 产出package.json的文件
4.新建 gulpfile.js
5.运行 npm install gulp gulp-jade gulp-sass
6.创建 sass 文件目录，jade文件目录
7.在gulpfile.js 引入插件
//引入 require（“引入的插件名称”）；
8.定制任务
//gulp.task(“人物名”，function(){
gulp.src(“文件路径”)
.pipe（插件方法）
.pipe（gulp.desk（”输出路径“））
})

这是gulp完整的步骤，如果大家想实时的更新在装个 watch， 文件引入的时候格式很上面一样的
